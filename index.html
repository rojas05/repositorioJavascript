    • El uso más común de JavaScript es escribir funciones embebidas o incluidas en páginas HTML y que interactúan con el Document Object Model (DOM o Modelo de Objetos del Documento) de la página. Algunos ejemplos sencillos de este uso son: Cargar nuevo contenido para la página o enviar datos al servidor a través de AJAX sin necesidad de recargar la página (por ejemplo, una red social puede permitir al usuario enviar actualizaciones de estado sin salir de la página). Animación de los elementos de página, hacerlos desaparecer, cambiar su tamaño, moverlos, etc. Contenido interactivo, por ejemplo, juegos y reproducción de audio y vídeo.
